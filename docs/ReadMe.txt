风险声明：
本软件基于pyqt5，识别模块使用easyocr，识别精度会受到各种因素的影响，请自行斟酌。

使用指南：
1、本品对屏幕分辨率以及模拟器的配置有要求：

操作系统：window10,window11可运行，win7未测试过。
显示器分辨率：1920 x 1080
模拟器：推荐使用MuMu模拟器12！目前没有在其他各类模拟器上运行过，如果您发现同样适配其他模拟器，欢迎联系我。
模拟器分辨率：
MuMu12：（手机版）900 x 1600 (DPI 320)

2、适用范围：
适用于“猫三国”或其同源异名系列游戏（如：喵星联盟）的兵符洗练
支持的兵符：天龙符，天龙符+1，天龙符+2，乾坤符

3、游戏内操作：
点开游戏武将培养界面，点击“兵符”，点击“养成”——“重铸”，对未洗练的兵符，请先进行一次手动洗练，确保右边的新兵符显示框有数据，同时点掉系统自带的洗练询问提示。

将模拟器窗口切至后台。

！！
不是最小化窗口！否则软件的读取会失败！
！！

4、软件操作：
新建兵符方案，输入方案后，保存。

方案名称：本方案的名称，名称不能重复，不能为空，尽量选择易于理解的名称。

主属性：兵符必须满足该方案勾选的主属性之一。

副属性：兵符中至少包含您指定次数的要求副属性
	example：2,['攻击']['暴击']['暴击伤害']：兵符的所有副属性中，至少有2个副属性为攻击，暴击，爆伤其中之一。
	warn：天龙，天龙+1只有1个副属性！软件只会确保最基本的不合法方案不会产生，请不要在洗+1的时候要求2个副属性！

必有技能：兵符中必须要求存在的技能
	warn：不要超过3，这是不合法的，会被阻止。
	warn：若您勾选了高级迅捷，即使刷出超级迅捷，也不会被认为满足要求，请自行把握。

可选技能：兵符中至少出现了X次您勾选的技能
	warn：X不要超过3，这是不合法的，会被阻止。
	example:1,['超级暴击']['超级爆伤']：兵符中至少包含超级暴击，超级爆伤中的1个技能。

排除技能：兵符中不允许存在的技能（只给了几个可能产生负面的技能）
	warn：如果你不打算睡觉的时候洗极品符，建议别点这玩意。


新建好方案后，请！务必！将其添加到应用列表，只有被应用的方案才会实际被投入使用。

选中方案并应用之，确保模拟器窗口在运行且没有最小化，然后就可以开始启动脚本了。

！！
模拟器窗口可以在后台，但不要把模拟器窗口最小化！否则洗符会直接中断，需要重新启动之。
！！
洗符过程中，请不要大幅度移动模拟器窗口！
！！
